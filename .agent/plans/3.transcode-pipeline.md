# Phase 3: FFmpeg Transcoding Pipeline

## Completed
- [x] `services/transcode.py` — FFmpeg HLS transcoding (360p/720p/1080p adaptive)
- [x] `services/thumbnail.py` — Thumbnail extraction at 2s mark
- [x] `workers/transcode_worker.py` — Background worker (polls queued → processing → ready/error)
- [x] SSE endpoint `GET /api/videos/status/stream` for real-time status
- [x] Frontend dashboard SSE integration (live status badge updates)

## Architecture

```
Upload → DB status: queued
                ↓
Worker polls DB (every 5s)
                ↓
Claims video (FOR UPDATE SKIP LOCKED)
                ↓
Download from Supabase Storage → /tmp
                ↓
ffprobe → get resolution/duration
                ↓
FFmpeg → HLS per applicable preset (360p/720p/1080p)
                ↓
Generate master.m3u8 (adaptive)
                ↓
Generate thumbnail.jpg
                ↓
Upload HLS + thumbnail to Supabase Storage
                ↓
DB status: ready (with duration, width, height, thumbnail_url)
```

## How to run

```bash
# Terminal 1: API server
cd backend && uv run uvicorn main:app --reload --port 8007

# Terminal 2: Transcode worker
cd backend && uv run python -m workers.transcode_worker

# Terminal 3: Frontend
cd frontend && npm run dev
```
